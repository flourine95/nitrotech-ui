# Product

## Register

product

## Users

NitroTech serves shoppers comparing technology products and staff managing catalog, inventory, orders, media, and access in an admin dashboard. Admin users work in dense table views and need fast scanning, reliable filtering, and low-friction repeated actions.

## Product Purpose

NitroTech UI helps customers browse, compare, cart, checkout, and track orders while giving staff practical tools to operate the store. Success means product and order information is easy to find, actions feel predictable, and dashboard state stays in sync with backend data.

## Brand Personality

Clear, practical, trustworthy.

## Anti-references

Avoid decorative SaaS dashboards, oversized marketing composition inside task screens, gradient-heavy surfaces, glassmorphism, and UI that hides operational data behind visual flourish.

## Design Principles

- Make dense information scannable before making it expressive.
- Keep admin interactions predictable and reversible.
- Use cached or optimistic UI only when backend truth can be restored quickly.
- Separate static filter options from changing data counts when it improves perceived speed.
- Preserve compact controls and table ergonomics for repeated staff workflows.

## Accessibility & Inclusion

Target accessible keyboard navigation, visible focus states, readable contrast, reduced-motion-safe transitions, and Vietnamese-friendly typography.
